# ⚡ AI Content Detector — Lightweight Version (B)

本版本為「AI 內容偵測器」的 **輕量快速展示版**，  
專為：

- ✅ Streamlit Cloud 線上展示
- ✅ 課堂即時操作
- ✅ 助教快速測試
- ✅ 避免大型模型下載失敗

所設計。

## ✅ 本版本特色

- 🚀 **秒啟動（無模型下載）**
- ❌ 不使用 PyTorch / GPT-2
- ✅ 純規則（Rule-based）+
- ✅ 統計特徵（Burstiness）
- ✅ 即時顯示 AI 機率 %
- ✅ 句子顏色標記
- ✅ 橫向長條圖分析
