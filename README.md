# AI Content Detector 🤖

一個基於 GPT-2 模型的 AI 內容檢測工具，可以分析文本並判斷其是否由 AI 生成。

## 功能特點

- 🌐 **雙語模型選擇**：支援英文模型 (GPT-2) 和中文模型 (GPT2-Chinese)，可根據文章語言自由切換
- 📝 **多種輸入方式**：支援直接貼上文字或上傳檔案（TXT、PDF、DOCX）
- 🎯 **AI 機率評分**：計算文本由 AI 生成的可能性百分比
- 📈 **Burstiness 評分**：分析句子長度變化，AI 生成的文本通常變化較小
- 🔍 **句子級別分析**：逐句標註並用不同顏色標示 AI 生成的可能性
- 📊 **數據視覺化**：橫向長條圖展示每個句子的 AI 可能性
- 🎨 **動態漸層背景**：現代化的使用者介面設計
- 🌏 **中英文支援**：支援中文和英文標點符號的句子分割

## 技術架構

- **前端框架**：Streamlit
- **AI 模型**：GPT-2 (from Hugging Face Transformers)
- **文檔處理**：PyPDF2 (PDF)、python-docx (Word)
- **數據視覺化**：Altair、Pandas
- **深度學習**：PyTorch、Transformers

## 安裝方法

1. 克隆或下載此專案

2. 安裝依賴套件：
```bash
pip install -r requirements.txt
```

## 使用方法

1. 啟動應用程式：
```bash
streamlit run main.py
```

2. 在瀏覽器中開啟顯示的本地網址（通常是 `http://localhost:8501`）

3. **選擇語言模型**（左側邊欄）：
   - **English (英文)**：適合分析英文文章，使用原版 GPT-2 模型
   - **Traditional Chinese (中文)**：適合分析中文文章，使用 GPT2-Chinese 模型

4. 輸入文本：
   - **方法一**：直接在文字框中貼上或輸入文字
   - **方法二**：點擊上傳按鈕，選擇 TXT、PDF 或 DOCX 檔案

5. 點擊「🔍 Start Analysis」按鈕開始分析

## 分析結果說明

### AI 可能性評分
- **80%以上**：高度疑似 AI 生成（紅色標記）
- **60-80%**：可能由 AI 生成（黃色標記）
- **60%以下**：較像人類撰寫（綠色標記）

### Burstiness Score（爆發性評分）
- **數值越高**：句子長度變化越大，越像人類寫作風格
- **數值越低**：句子長度較一致，可能是 AI 生成
- 人類寫作通常會有長短句交替，AI 則傾向於產生長度相似的句子

### 技術原理

#### 1. Perplexity（困惑度）分析
本工具使用 **Perplexity** 來判斷文本的 AI 生成可能性：
- AI 生成的文本通常具有較低的困惑度（更可預測）
- 人類撰寫的文本通常具有較高的困惑度（更有創意和變化）

#### 2. 雙語模型支援
- **英文模型 (GPT-2)**：針對英文語料訓練，適合分析英文文章
- **中文模型 (GPT2-Chinese)**：使用 CLUECorpus 中文語料訓練，更準確判斷中文文本

## 支援的模型

### 英文模型
- **模型名稱**：GPT-2
- **模型 ID**：`gpt2`
- **適用語言**：英文
- **模型大小**：約 500MB

### 中文模型
- **模型名稱**：GPT2-Chinese (CLUECorpusSmall)
- **模型 ID**：`uer/gpt2-chinese-cluecorpussmall`
- **適用語言**：繁體中文、簡體中文
- **模型大小**：約 500MB次執行時會自動下載 GPT-2 模型（約 500MB），請確保網路連線穩定
- 分析結果僅供參考，不應作為唯一的判斷依據
- 較短的文本可能會影響準確度

## 常見問題

**Q: 為什麼第一次啟動很慢？**  
A: 首次執行需要從 Hugging Face 下載 GPT-2 模型，之後會使用快取，速度會快很多。每個模型第一次使用時都需要下載。

**Q: 支援哪些語言？**  
A: 
- **英文**：使用原版 GPT-2 模型，準確度高
- **中文**：使用 GPT2-Chinese 模型，專門針對中文優化
- 請根據文章語言在左側邊欄選擇對應的模型以獲得最佳結果

**Q: 可以離線使用嗎？**  
A: 下載模型後可以離線使用，但首次執行必須連網。每個模型（英文/中文）都需要單獨下載一次。

**Q: 切換模型後會重新下載嗎？**  
A: 不會。模型下載後會儲存在本地快取中，切換模型時會直接從快取載入，速度很快。

**Q: 中文模型支援繁體中文嗎？**  
A: 是的，中文模型同時支援繁體和簡體中文。

**Q: Burstiness Score 怎麼看？**  
A: 這個分數代表句子長度的變化程度。人類寫作通常會有長短句交替（高分），而 AI 生成的文本句子長度較為一致（低分）。
